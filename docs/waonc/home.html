Home Page

 

This page describes the Waonc project, which is a fork of WaoN, a project that converts audio files to MIDI files.

 

Waonc is a minor refactoring of the WaoN codebase. See the INSTALL file. We're still working on improvement the detection of the many libraries needed for this project; for now, it builds and runs just fine. A short manual is included:

 
  • PDF Quick Reference Manual A wave-to-MIDI converter.
 

As time goes on, more to do! See https://github.com/ahlstromcj/waonc
